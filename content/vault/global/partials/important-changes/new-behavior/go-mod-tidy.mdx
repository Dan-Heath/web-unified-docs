### Go mod tidy failures on community edition ((#go-mod-tidy))

<!-- BEGIN: Vault:=v1.20.x -->

| Change       | Affected version | Vault edition
| ------------ | ---------------- | -------------
| New behavior | 1.20.4           | Community

<!-- END: Vault:=v1.20.x -->

The `go.mod` file lists a private repo that causes the `go mod tidy` command to
fail.

#### Recommendation

Update the `go.mod` file to remove the line referencing `hashicorp/go-cmp` or
build from `main`.
